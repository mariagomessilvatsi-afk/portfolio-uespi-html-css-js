# Meu Portfólio

Estrutura de pastas organizada para o projeto do portfólio.

## Estrutura
- `assets/` — arquivos estáticos (imagens, CSS, JS, fontes)
  - `img/` — imagens
  - `css/` — estilos
  - `js/` — scripts
  - `fonts/` — fontes
- `pages/` — páginas HTML adicionais
- `components/` — componentes reutilizáveis (ex: navbar, footer)
- `docs/` — documentação do projeto
- `index.html` — página principal
